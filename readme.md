# Comandos Terminal

### Listar arq e pastas
```
ls
```

### Abrir VS na pasta
```
code .
```

### Criar pasta
```
mkdir NOME_DA_PASTA
```

### Acessar a pasta
```
cd NOME_DA_PASTA
```

### Criar E editar arq
```
nano NOME_DO_ARQUIVO
```

### Remover pasta e arquivos internos
```
rm -rf NOME_DA_PASTA
```
